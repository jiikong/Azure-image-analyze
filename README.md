# Azure Image Analysis Demo

Azure Computer Vision 서비스를 활용한 이미지 분석 및 검색 데모 프로젝트입니다.  
이미지로부터 캡션, 태그, 객체 정보를 추출하고, 분석 결과를 JSON 형태로 저장하여  
이미지 검색 및 시각화 데모에 활용하는 것을 목표로 합니다.

---

## 📌 프로젝트 개요

본 프로젝트는 Azure Computer Vision API를 사용하여 다음 기능을 구현합니다.

- 이미지 캡션(Caption) 생성
- 태그(Tag) 및 객체(Object) 분석
- 분석 결과 JSON 저장
- 분석 데이터를 활용한 검색 및 데모 애플리케이션 구성

본 저장소는 **API Key 보안 문제를 고려하여 환경변수 방식으로 구성**되어 있습니다.

---
